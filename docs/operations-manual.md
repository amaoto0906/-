# 養魚場管理システム 操作マニュアル

## 1. はじめに

このシステムは養魚場の日次業務（給餌・生産・水質）を記録・集計・帳票化するWebアプリです。
PC / タブレットの主要ブラウザ（Chrome / Edge / Safari / Firefox）で動作します。

---

## 2. ログインと権限

### ログイン方法

1. ブラウザで http://（サーバーアドレス）/login を開きます。
2. メールアドレスとパスワードを入力して「ログイン」を押します。
3. ダッシュボードが表示されればログイン成功です。

### 権限ロール

| ロール | 主な操作 |
|---|---|
| 管理者 (admin) | 全機能 + マスタ管理（追加・編集・無効化・削除）+ ユーザー削除 |
| 作業員 (worker) | 給餌・生産・水質の入力／編集／削除、自分のパスワード変更 |
| 閲覧者 (viewer) | 一覧と帳票の閲覧のみ |

### パスワード変更

1. 左メニュー「アカウント設定」を開く
2. 「🔑 パスワード変更」を押す
3. 現在のパスワード、新パスワード、確認用パスワードを入力して保存

---

## 3. ダッシュボード

ログイン後の起点画面です。

- **期間タブ**：本日／今週／今月 を切り替えると、池別状況・魚種別状況・KPIが切り替わります
- **KPIカード**：月間給餌量、生産量、死亡数、FCR（飼料係数）
- **30日の推移グラフ**：給餌量・生産量・死亡数の折れ線
- **アラート**：未確認の異常検知をプレビュー（クリックで詳細）
- **池・水槽別状況**：池ごとの直近の給餌・生産・FCR・水温
- **魚種別状況**：魚種ごとの出荷尾数・重量・死亡数
- **クイックアクション**：給餌入力・生産入力・水質入力へのショートカット

---

## 4. 日次入力

### 給餌入力

「給餌入力」または ダッシュボードの🍚カードを押す。

1. 日付（既定: 本日）
2. 池・水槽（選択）
3. 飼料（選択）
4. 給餌量 (kg) — 0より大、上限 10,000kg
5. 給餌回数 — 1〜20回
6. 備考（任意）
7. 「記録する」で保存。記録後は自動で集計・異常検知を再実行します

### 生産入力

「生産入力」または ダッシュボードの🐟カードを押す。

1. 日付（既定: 本日）
2. 池・水槽 / 魚種
3. 出荷尾数 / 出荷総重量 (kg)
4. 死亡数
5. 平均体重 (g) — 任意
6. 備考
7. 「記録する」で保存

### 水質入力

「水質入力」または ダッシュボードの💧カードを押す。

1. 日付
2. 池・水槽
3. 水温（℃） / pH / DO（mg/L）— いずれか入力
4. 備考
5. 「記録する」で保存

## 5. 履歴・検索

「給餌履歴」「生産履歴」では以下の条件で絞り込めます。

- 池・水槽
- 魚種（生産履歴）／飼料（給餌履歴）
- 期間（開始日〜終了日）
- キーワード（備考／池名／魚種名／飼料名）

結果はページネーション（1ページ 20件）で表示されます。各行から「編集」または「削除」が可能です。
削除時は確認ダイアログで再確認します。

---

## 6. 帳票（日報・週報・月報）

「帳票」メニューで日報／週報／月報を切り替えます。

各行の操作：
- **プレビュー**：画面上で内容を確認
- **PDF**：PDFファイルをダウンロード

PDFには以下が含まれます：
- 期間サマリー
- AIコメント（前日比・前週比・前月比、アラート要約）
- 池別集計（給餌量・生産量・死亡数・FCR）
- 魚種別集計（出荷尾数・重量・死亡数）
- 給餌記録一覧（日報のみ）
- 生産記録一覧（日報のみ）

ファイル名: `daily-report-YYYY-MM-DD.pdf` / `weekly-report-YYYY-Www.pdf` / `monthly-report-YYYY-MM.pdf`

---

## 7. 異常検知アラート

「アラート」メニューで未確認・確認済みのアラートを一覧できます。
ダッシュボード表示時に自動で異常検知が実行され、新規アラートが記録されます。

検知ルール：

| 種別 | 検知条件 |
|---|---|
| 給餌量スパイク | 直近7日平均から ±2σ を超える |
| 死亡数スパイク | 過去30日中央値の3倍以上 |
| FCR異常 | 魚種ごとの基準レンジ外 |
| 入力欠損 | 直近3日間の給餌記録なし |
| 生産量急減 | 直近2日が過去平均の30%以下 |

「確認」ボタンで未確認 → 確認済みに変更できます。

---

## 8. 簡易予測

「予測」メニューで給餌量の予測を確認できます。

- 池・水槽を選択
- 予測期間（7日 / 14日 / 30日）を選択
- 「更新」を押す

2モデルを併記：

- **移動平均モデル**：直近平均値で平坦に予測（青破線）
- **線形トレンドモデル**：最小二乗法で傾向を推定（緑破線）

⚠️ データが5日未満の場合は警告が表示され、予測値の信頼性は限定的です。

---

## 9. マスタ管理（管理者のみ）

「池・水槽」「魚種」「飼料」の各マスタを管理します。

操作：
- **新規追加** → コード・名称・属性を入力して保存
- **編集** → 既存レコードを変更
- **無効化／有効化** → 入力フォームの選択肢から外す（過去レコードは残る）
- **削除** → レコードと関連データをまとめて削除（確認ダイアログあり）

無効化したマスタはグレー表示され、給餌・生産・水質入力フォームの選択肢には現れません。

---

## 10. 設定

- **アカウント情報** → 名前・メール・ロールの確認
- **パスワード変更** → 現在のパスワード確認 + 新パスワード設定

---

## 11. トラブルシューティング

| 症状 | 対処 |
|---|---|
| ログインできない | パスワード再確認。それでもダメな場合は管理者へ |
| アラートが古い | ダッシュボードを開くと自動で再検出されます |
| PDFが日本語化け | 既知の問題なし。再度ダウンロードを試してください |
| ページが真っ白 | 「再試行」ボタンが表示される場合があります。それでもダメな場合はブラウザを再読み込み |
| 404エラー | URLが正しいか確認。ダッシュボードに戻って導線から再アクセス |
